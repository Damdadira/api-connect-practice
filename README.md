# API Connect

### 📚 학습 내용
- `Axios`를 사용하여 비동기 API 호출을 구현
- `useAsync`라는 커스텀 훅을 만들어 API 요청, 응답 및 에러 상태를 관리

### 🎯 실행 방법

- **Node.js 18 이상 권장**

```bash
# 패키지 설치
npm install   # 또는 npm i

# 개발 서버 실행
npm run dev
```
